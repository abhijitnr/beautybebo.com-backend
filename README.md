# BEAUTYBEBO.COM BACKEND

## About

- This is the backend of beautybebo.com
- In this backend you can get the follwing routes or endpoint
  - signup
  - login
  - get all users
  - get user by id
  - delete user
  - get all products
  - get product by id
  - update product details
  - delete product
